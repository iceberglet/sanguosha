# Node版 三国杀 服务器程序


### 使用方法

Dev 安装
```cmd
    npm install
    cd ./resources
    node server.js #可以直接用repo的开始游玩
```

设置位于:  ```/resources/config-server.json```


### 须知

- 当前仅支持国战模式
- 无AI
- 暂时最多八人游戏
